# Guava

## About Guava

:rainbow: [Guava](https://marketplace.visualstudio.com/items?itemName=ConAntares.Guava) is made for [VS Code](https://code.visualstudio.com/). It contains both dark theme and light theme. Its soft colors will be helpful for you to get inspired!


## Installation

1. Launch Quick Open  

   * Linux:    `Ctrl` + `P`
   * macOS:    `⌘ P`
   * Windows:  `Ctrl` + `P`

    Paste the following command and press Enter:  
        ```ext Guava```

    Or install it via sidebar panel in VS Code: Open Extension sidebar panel, and Search for "Guava";

2. Click Install to download and install it;
3. Click Reload to reload your VS Code;
4. Activate it:  
    `Manage` → `Color Theme` → `Guava`
5. Enjoy it !
